# AI数据处理分析师学习指南

## 📖 快速开始

### 1. 环境准备

```bash
# 安装必要的包
pip install -r requirements.txt

# 如果需要中文分词
pip install jieba

# 如果需要可视化
pip install matplotlib

# 如果需要机器学习
pip install scikit-learn torch
```

### 2. 数据集下载

首先运行 `get_jd_datasets.py` 下载并预览数据集：

```bash
python get_jd_datasets.py
```

### 3. 开始学习

按照 `data_processing_exercises.md` 中的练习顺序，从阶段一开始：

1. **阶段一**：数据下载与加载（pandas基础）
2. **阶段二**：数据探索与统计（pandas进阶）
3. **阶段三**：数据清洗（pandas实战）
4. **阶段四**：数据可视化（matplotlib）
5. **阶段五**：文本预处理（jieba）
6. **阶段六**：特征工程（pandas + numpy）
7. **阶段七**：数据标注与增强（pandas）
8. **阶段八**：数据预处理（为模型准备）
9. **阶段九**：简单文本分类模型（PyTorch）

### 4. 使用代码模板

`exercise_templates.py` 提供了每个练习的基础代码框架，你可以：
- 参考模板理解任务要求
- 完成TODO部分
- 运行并验证结果

## 📝 学习建议

1. **循序渐进**：不要跳步，每个阶段都要扎实掌握
2. **动手实践**：一定要亲自写代码，不要只看
3. **理解原理**：不仅要会写，还要理解为什么
4. **记录笔记**：遇到问题及时记录
5. **查阅文档**：学会使用官方文档解决问题

## 🎯 每个阶段的学习目标

- **阶段一**：掌握pandas基础操作
- **阶段二**：学会数据探索和分析
- **阶段三**：掌握数据清洗方法
- **阶段四**：学会数据可视化
- **阶段五**：掌握文本预处理
- **阶段六**：理解特征工程
- **阶段七**：处理数据不平衡
- **阶段八**：为模型准备数据
- **阶段九**：训练简单分类模型

## 💡 遇到问题？

2. 参考 `exercise_templates.py` 中的代码
3. 查阅相关包的官方文档
4. 记录问题，逐步解决
